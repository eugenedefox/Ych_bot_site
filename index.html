✦ How it works

Simple publishing flow

The project focuses on a clear Telegram workflow for artists who want fewer tabs, fewer copy-paste mistakes, and more consistent auction posts.

1 Upload art Send an image to the bot and let it prepare a draft.
2 Edit details Adjust title, text, tags, template, and auction link.
3 Pick targets Enable only the platforms needed for this post.
4 Publish Send the final post after checking the preview.

✦ Platform integration

Instagram API use case

The bot uses Instagram API access only to let an artist publish artwork to an Instagram Professional account that the artist explicitly connects.

What the integration does

  • Connects a user's own Instagram Professional account through OAuth.
  • Creates an image media container from a public temporary image URL.
  • Publishes the post only after the user presses Publish in Telegram.
  • Adds the caption, tags, and auction text prepared by the user.

Requested permissions

  • instagram_business_basic identifies the connected account.
  • instagram_business_content_publish publishes artwork posts.
  • The bot does not use Instagram messages, comments, ads, or analytics.
  • The bot does not publish automatically or on a schedule without user action.

✦ Your data

Privacy policy

YCH Commishes Bot stores only the information needed to provide publishing, auction tracking, and account connection features.

Data the bot may store

  • Telegram user ID and username for bot access and settings.
  • Connected platform tokens for services the user chooses to connect.
  • Publishing settings, templates, tags, auction links, and channel IDs.
  • Temporary media files or public media URLs required for publishing.

How data is used

  • To publish artwork only after the user confirms the post.
  • To remember the user's connected platforms and posting preferences.
  • To track YCH auction links and notify about bid-related updates.
  • Data is not sold, rented, used for advertising, or used for profiling.

✦ Account management

Data deletion

Users can disconnect integrations directly in Telegram. For full deletion, contact the bot administrator with the Telegram username or Telegram user ID.

Instagram data deletion: send /disconnect_instagram to remove the stored Instagram token from the bot. To request full account data deletion, email eugenedefox@gmail.com with your Telegram username and the request "Delete my YCH Commishes Bot data".

✦ Background

Why it exists

YCH Commishes Bot is made as a practical helper for managing artwork publishing and auction communication. It is intended to support artists with repetitive posting, bid-link organization, and consistent descriptions across platforms.

This project is an independent automation tool. It is not an official Tumblr, Instagram, DeviantArt, Bluesky, Discord, Telegram, or YCH Commishes product.
